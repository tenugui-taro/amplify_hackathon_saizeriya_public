# サイゼリヤ 1000 円ガチャ

## 実行方法

1. ファイル _amplify_hackathon_saizeriya.py_ の、`client.token`のコメントアウトを外してアクセストークンを入力する

2. _amplify_hackathon_saizeriya.py_ を実行する

```shell
$ python amplify_hackathon_saizeriya.py
```
